# Week 4 - Image Text Recognition (OCR)

## Project Overview
This project performs Optical Character Recognition (OCR) to extract text from images using Python.

## Features
- Reads text from images
- Uses OpenCV for image processing
- Uses Pytesseract OCR engine
- Displays extracted text in the terminal

## Technologies Used
- Python
- OpenCV
- Pytesseract
- Pillow
